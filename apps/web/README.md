## Web app notes

This workspace is the dashboard and API surface for the Coachin'Claw monorepo.

- Local dev: `npm run dev:web` from the repository root
- Production-style start: `npm run start --workspace @coachinclaw/web -- --hostname 0.0.0.0 --port 3000`
- Self-hosted stack: use the repo-root `docker-compose.yml`

For deployment and self-hosting instructions, use the repository root `README.md` rather than this workspace file.
